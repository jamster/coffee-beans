A CoffeeScript console. Here's the source.

Underscore functions are available in the global namespace. Some examples: 

range 10                         # create a range

isNumber 5                       # is 5 a number (it is)

size {a:1, b:2}                  # size of object or array

keys Backbone                    # get Backbone's properties

sort uniq [4,2,8,4,1,2]          # unique elements, sorted

map [1,2,3], (x) -> x*x          # square some numbers

The code editor is on the bottom left. These are the keyboard shortcuts:

  • Enter: Run command
  • Shift+Enter: New line
  • Ctrl+Space: Autocomplete
  • Tab: Indent
  • Up: Previous command
  • Down: Next command

Command history is saved in local storage, so you can access past commands when you come back.

Basics

Arrays

five = [1,2,3,4,5]

Objects

fruit_colors =
  apple: 'red'
  banana: 'yellow'
  kiwi: 'green'

Functions

square = (x) -> x*x

Comprehensions

output x*x for x in [1,2,3]

Chained Comparisons

age = 14
teen = 12 < age < 20

Functional Programming

Each

Perform a function on each element of an array or object

each ['here', 'we', 'go!'], alert

Map

Convert radians to degrees

pi = Math.PI

angles = [pi/4, 3*pi/8, pi/2]      # data
convert = (x) -> x*180/pi          # function to map

map angles, convert

Filter

Filter even numbers

numbers = [1,2,3,4,5,6]            # data
even = (x) -> x % 2 == 0           # filter function

filter numbers, even
